# Lumix-website
O website da Lumix.
Aqui é onde os usuários vão poder criar suas "contas Lumix", para fazer votação, comentar, favoritar em jogos, etc.
Os novos usuários poderão encontrar as contas nas redes sociais, poderão entrar no grupo do discord feito exclusivamente pros fãs, etc.
É aqui onde terá as principais notícias sobre os jogos como recordes dos players, jogos novos, bugs arrumados ...

observação: esse site está em desenvolvimento, portanto, atualizará constantemente.
